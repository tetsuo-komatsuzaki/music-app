## 著作権に関する注意事項

1. **楽譜の版について**: 作曲者の著作権が切れていても、特定の編集者による校訂版（ボウイング・指使い等を加えたもの）には編集著作権が存在する場合があります。
2. **推奨楽譜ソース**: [IMSLP (International Music Score Library Project)](https://imslp.org/) の原典版（Urtext）を推奨。
3. **編曲版**: 上記リスト中、編曲版（クライスラー編等）の場合は編曲者の著作権にも注意が必要です。個別に記載しています。
4. **演奏録音**: 楽譜の著作権と演奏の著作権は別です。録音には演奏者・レコード会社の著作隣接権があります。
